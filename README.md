### 🚗 Traffic Volume Prediction Dashboard — sklearn Pipelines + Interactive Analytics
A **production-quality ML dashboard** that predicts traffic volume using **sklearn Pipelines** + automated preprocessing and feature engineering.  
Built to eliminate common production bugs like **feature mismatch** and failures on **unknown categories** by using robust pipeline design (e.g., OneHotEncoder with `handle_unknown="ignore"`).  
Includes **3 trained models** (Linear Regression, Decision Tree, Random Forest) and displays live evaluation metrics (MSE/RMSE/MAE/R²) with interactive visualizations.  

🔗 Repo: https://github.com/SoubhagyaJain/Traffic-prediction

---

### 🎬 Collaborative Filtering Movie Recommender — User-User CF + Streamlit App
A **user-user collaborative filtering** movie recommender that:
- analyzes user ratings,
- finds similar users via **cosine similarity**,
- recommends movies a user hasn’t watched based on similar users’ preferences,
- and serves results through a **Streamlit** UI.

🔗 Repo: https://github.com/SoubhagyaJain/Collabrative-Filtering-System
